이 프로젝트는 **피부톤, 성별, 나이, 선호 브랜드** 정보를 기반으로  
사용자에게 적합한 화장품을 추천하는 시스템입니다.  
Pandas와 Scikit-learn을 활용해 데이터 전처리 및 추천 알고리즘을 구현했으며,  
웹 페이지에서는 Hugoblox를 활용해 사용자가 입력한 데이터를 시각적으로 피드백 받을 수 있도록 구성했습니다.
